# 家計簿アプリ — Claude編集ガイド

## このプロジェクトについて
iPhoneにPWAインストールして使う個人用家計簿アプリ。
**すべての機能は `index.html` 1ファイルに集約されている。**

---

## ファイル構成

| ファイル | 用途 | 編集頻度 |
|---|---|---|
| `index.html` | メインアプリ本体（1814行） | ★★★ 主な編集対象 |
| `kakeibo_mock.html` | モック/デモ版（実データなし） | ★☆☆ 参照用 |
| `sw.js` | Service Worker（オフラインキャッシュ） | ★☆☆ キャッシュバージョン更新時のみ |
| `manifest.json` | PWA設定 | ★☆☆ アイコン・テーマ変更時のみ |
| `dexie.min.js` | IndexedDB ORM（触らない） | 変更不要 |

---

## タブ構成

| タブ | screen ID | 説明 |
|---|---|---|
| ホーム | `sc-home` | 残高・収支サマリー・ドーナツグラフ・最近の取引 |
| カレンダー | `sc-cal` | 月次カレンダー・日別パネル |
| レポート | `sc-rep` | 月次/年間/サブスク/投資の4サブタブ |
| 設定 | `sc-set` | 予算・カスタムカテゴリ・データ管理 |

---

## 主要なグローバル変数

| 変数 | 型 | 内容 |
|---|---|---|
| `TXS` | Array | 全取引データ（IndexedDBから起動時ロード） |
| `CC` | Array | カスタムカテゴリ |
| `DC` | Array | デフォルトカテゴリ（変更不可） |
| `SUBS` | Array | サブスクリプション |
| `BUDGET` | Number | 月次予算（デフォルト300000） |
| `S` | Object | UIステート（タブ・入力値・選択中の日付など） |

## よく使う関数

### 画面更新
```js
rHome()         // ホーム画面を再描画
rCal()          // カレンダーを再描画
rMonthly()      // 月次レポートを再描画
```

### 取引操作
```js
saveTx()        // 入力フォームの内容を保存
editTx(id)      // 既存取引を編集モードで開く
delTx(id)       // 取引を削除
```

### ユーティリティ
```js
fmt(n)          // → "¥ 1,234" 形式にフォーマット
fmtC(n)         // → "1.2万" 形式（コンパクト表示）
nowYM()         // → "2026-05" 形式の現在年月
todayStr()      // → "2026-05-25" 形式の今日の日付
toast("msg")    // 画面下部にトースト通知を表示
gc(id)          // カテゴリIDからカテゴリオブジェクトを取得
```

### DB操作
```js
dbAdd(tx)               // 取引を追加
dbPut(tx)               // 取引を上書き保存
dbDel(id)               // 取引を削除
dbSaveSetting(key, val) // 設定を保存
```

## 注意事項

1. **Service Workerのキャッシュ** — `index.html` を大幅変更した場合、`sw.js` の `CACHE_NAME: 'kakeibo-v33'` のバージョン番号を上げないと古いキャッシュが残る場合がある。
2. **単一ファイル構成** — CSS・JS・HTMLはすべて `index.html` に含める。外部ファイルに分割しない。
3. **デモ用フレーム** — デスクトップで開くとiPhoneフレームが表示される。`@media (display-mode: standalone)` でPWAインストール後の表示を制御している。
4. **IndexedDB** — データはブラウザのローカルストレージ（IndexedDB）に保存される。Dexieライブラリ経由でのみアクセスすること。
